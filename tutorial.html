Tutorial Hosting Web di GitHub

Profil | Tutorial

Langkah-langkah:

  1. Buat akun di GitHub.
  2. Buat repository baru dengan nama username.github.io.
  3. Upload file website kamu (HTML, CSS, gambar, dll).
  4. Setelah itu, akses web kamu di: https://username.github.io.

Selesai! Website kamu kini online gratis di GitHub Pages.
